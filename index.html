rust-analyzer

Bringing great IDE experience to the Rust programming language.

Source

Installation

rust-analyzer is in the alpha stage, installation requires building from source. To install the language server and VS Code plugin, use 

$ git clone https://github.com/rust-analyzer/rust-analyzer && cd rust-analyzer
$ cargo xtask install

See the manual for alternative installation methods and support for other editors.
